Gravitational Wave Viewer Translation Page

The form below contains all the language fields used in the Gravitational Wave Viewer (see the viewer in ).

The fields are split into sections. If you'd like to start translating a language that isn't on the drop down list please contact Chris North.

If you want your changes to be applied you'll have to copy/paste the output from the bottom into an email to Chris (chris.north AT astro.cf.ac.uk). This form does not automatically save!

If the language you would like is not available in the list, then start from another language, change the language code and name (the first two options) and send to Chris.

This method of generating translation files is adapted (but only slightly) from the method developed by Stuart Lowe for the Star in a Box Translation page.
